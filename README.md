# Tugas Akhir - Redesign Aplikasi Pemesanan Tiket Kapal

Repository ini dibuat untuk memenuhi kebutuhan Tugas Akhir pada Program Studi Informatika, Universitas Udayana.

## Informasi Tugas Akhir

**Judul Penelitian**
Redesign Aplikasi Pemesanan Tiket Kapal dalam Meningkatkan Pelayanan Berbasis User Centered Design

**Program Studi**
Informatika

**Universitas**
Universitas Udayana

## Deskripsi

Proyek ini berfokus pada perancangan ulang UI/UX aplikasi pemesanan tiket kapal berbasis mobile menggunakan metode User Centered Design (UCD). Penelitian ini dilakukan untuk menghasilkan rancangan aplikasi yang lebih mudah digunakan, informatif, dan sesuai dengan kebutuhan pengguna.

Hasil rancangan kemudian diimplementasikan ke dalam aplikasi Android sebagai bagian dari proses penelitian dan pengujian tugas akhir.

## Catatan

Repository ini digunakan untuk keperluan akademik. Data yang digunakan pada aplikasi merupakan data dummy dan tidak terhubung dengan sistem resmi perusahaan pelayaran mana pun.
